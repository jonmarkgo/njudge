Njudge mail format



Version 0.1 Christophe Courtois - 12nd September 2003 : first draft



Introduction

This document describes the output format of the njude Diplomacy adjudicator.


Goals

  1. Describe the current mail format to make compatibility between the judge and the tools which use its output easier.

  2. Help the next version of njudge to be fully compatible with the previous versions (compatibility mode).

  3. Describe the enhancements to this format in the new versions of njudge or other judges.

Existing mail types

The following mails can be sent by the judge:

      • Press to players

      • Results of turns

      • Players is/are late or abandoned

      • Version
